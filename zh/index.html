常见问题

如果您找不到想要的答案，请直接联系我们。

与 NovaSQC 合作的流程是什么？
共有6个步骤：

1. 客户向 NovaSQC 发送订单详情和要求
2. 客户确认 NovaSQC 提供的费用
3. NovaSQC 与供应商/工厂协调安排
4. 合格的检验员前往现场进行实地检验
5. 出具带详细检验结果的图文报告
6. 客户决定下一步：返工、复检、出货或拒收
标价之外还有其他费用吗？
没有。您在我们网站上看到的所有价格都是全包价格。没有隐藏费用。它涵盖了所有相关费用，如检验员时间、现场差旅、用餐和检验报告。
我的供应商推荐了他们自己的检验公司。我应该用吗？
我们强烈建议不要这样做。检验公司应该是独立的第三方。供应商只会推荐他们有关系的检验公司——这意味着检验员很可能会让不良订单通过以维护这种关系。NovaSQC 只为客户——您——工作。
你们的检验员持有哪些认证？
我们的检验员持有国际认可的资质，包括 API 1169、CWI、NACE 和 ISO 9001。所有检验员至少有4年以上现场经验。各地认证情况不同——请联系我们了解您所在地区的具体检验员资质。
你们的检验员会说哪些语言？
我们的检验员会说英语、法语、葡萄牙语、阿拉伯语、斯瓦希里语、豪萨语、阿姆哈拉语、绍纳语、祖鲁语等。我们会根据您的项目地点匹配检验员，确保与所有相关方的清晰沟通。
我可以取消或重新安排检验吗？
可以。您可以在预定日期前一天当地时间下午4点前免费取消或重新安排检验。如需紧急重新安排，请直接联系我们，我们会尽力满足您的需求。
如果对检验不满意怎么办？
不满意，全额退款。如果您对检验质量不满意，我们将免费重新检验或提供全额退款。您的满意度和项目完整性是我们的首要任务。
